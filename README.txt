
【操作(英文输入法)】
    -- 按键'上'  --灯光变亮； 
    -- 按键'下'  --灯光变暗；
    -- 按键'左'  --视角左转； 
    -- 按键'右'  --视角右转；
    -- 按键'W'  --视角前进； 
    -- 按键'S'   --视角后退；
    -- 按键'A'  --视角左移； 
    -- 按键'D'  --视角右移；
    -- 按键'Z'  --茶壶倾斜； 
    -- 按键'X'  --茶壶归位；
    -- 按键'C’--椅子变高； 
    -- 按键'V'  --椅子变矮；
    -- 按键'B'  --开/关门；
